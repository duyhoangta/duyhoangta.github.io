Git for dummies like me

Installation: Check this link.

Clone a respository

  1. Get the clone link:

    The link would look something like this: https://github.com/hunglvosu/subsetTSP.git
  2. Creat a local folder, point to it and type git clone https://github.com/hunglvosu/subsetTSP.git. A folder name "subsetTSP" will be created like in the following figure:

  3. Now you are ready to make changes, add files or do whatever you want to the folder. This is just local change, and nobody can see your change except you, unless you "commit" the change following the instructions below.
After cloning and make changes, you need to commit changes to the respository so other people can see it.

Commit changes

Note: Before making any changes, you should pull the most updated version of the repository to your local by typing: git pull.
  1. Type git status to see changes that you have made. In the following figure, I have made change to the file name subset-tsp.tex in my local repository.

  2. Type git add . to add all the changes for committing. Note that the dot is importan to have. This essentially tell git that "I've added changes, now I ready to commit the changes". If you don't want to add all changes, but just some files, say a.tex and b.tex, you can type git add a.tex b.tex instead.
  3. Type git commit -m "Message to describe the change" to commit. This tells gits that "I'm committing to my change". Still, nobody sees the change yet, except you. In the message, you can write whatever you want, but it should be something to remind you of what you have done, just in case you want to revert the change later in the distant future.
  4. Type git push origin HEAD to push the changes to the online repository. After this, everybody will see the change. No need to do anything else.

Add files to the repository

  1. Simply copy the files to your git local repository.

  2. Type git status to see the files you have added. In the following figure, I add two files to my local git repository.

  3. Type git add . to add all the changes for committing. If you just want to add some of the files, then you need to type all the file names instead of dot, as in commiting changes.
  4. Type git commit -m "dd two files latex8.sty and latexmac.sty" to commit.
  5. Type git push origin HEAD to push the change to the online repository.
